# Barbara Liskov

**Barbara Liskov** é uma cientista da computação norte-americana conhecida por suas contribuições fundamentais para a programação orientada a objetos, sistemas distribuídos e linguagens de programação.

Ela é especialmente famosa por formular o **Liskov Substitution Principle (LSP)**, um dos princípios que compõem o **SOLID**.

Barbara Liskov foi uma das primeiras mulheres a alcançar grande reconhecimento na área de ciência da computação.

Ela foi uma das pioneiras no conceito de **abstração de dados**, que é um dos pilares da orientação a objetos.

# O Princípio de Substituição de Liskov (LSP)

O **Liskov Substitution Principle** foi apresentado em 1987 em um artigo chamado:

> *Data Abstraction and Hierarchy*

O princípio afirma:

> Objetos de uma subclasse devem poder substituir objetos da classe base sem alterar o comportamento esperado do programa.

Em outras palavras:

Se um sistema funciona com uma classe base, ele também deve funcionar com qualquer uma de suas subclasses.

---

# Relação com SOLID

O LSP se tornou um dos princípios do **SOLID**, conjunto de boas práticas de design orientado a objetos.

- S — Single Responsibility Principle
- O — Open/Closed Principle
- L — Liskov Substitution Principle
- I — Interface Segregation Principle
- D — Dependency Inversion Principle


O **L** do SOLID vem diretamente do trabalho de Barbara Liskov.
